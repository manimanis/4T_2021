  • mot3 := copy(mot1, 3, 6);

  • mot4 := mot3 + mot2;
  • mot4 := mot3 + ' ' + mot2;
Question 3

Mettre dans n la longueur de la chaine mot4 :

  • n ← long(mot4)
  • n := length(mot4);
  • n := length(mot2) + length(mot3) + 1;
Question 4

  • m := Val(mot2[1]);
  • Valeur(mot2[1], m, e);
  • Val(mot2[1], m, e);
